## Architecture Principles

**Input code must be shared between creation and editing flows.** Any input/selection
logic (fuzzy select, prompts, validation) should use the same functions for both
creating new records and editing existing ones. This ensures consistent UX and
avoids bugs where fixes only apply to one flow.
